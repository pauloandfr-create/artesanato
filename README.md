# Artesanato App

Aplicativo Flutter para gerenciamento de produtos de uma loja de artesanato.

## Sobre o projeto

Este projeto permite:
- cadastrar produtos;
- editar informações e imagens;
- visualizar os itens em uma lista organizada;
- definir a logo da loja.

## Como rodar

1. Instale o Flutter.
2. Execute:
   ```bash
   flutter pub get
   flutter run
   ```

## Tecnologias

- Flutter
- Dart
- image_picker
